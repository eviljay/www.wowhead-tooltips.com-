Version 3.5 Released!

September 8th, 2009 | Tags: release, tooltip script
Version 3.5 of the script has been released. There were 2 issues fixed, as well as the addition of a script to automatically add the bbcode tags to phpBB3’s custom bbcode, which will add the buttons to the posting page.

Files Changed
  • table_scheme.sql
  • create_bbcode.php *new*
  • ./includes/phpArmory.class.php

Upload/overwrite the above files and then run create_table.php to successfully update the script to version 3.5.
